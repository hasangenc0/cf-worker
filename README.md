# Cloudflare Worker

```
 cargo install wrangler
 npm install
```

### Development
Run the following commands in separate shell sessions. 

```
npm run build:dev
```

```
npm run start:dev
```

### Build & Deploy
Run the following commands in separate shell sessions. 

```
npm run build
```

```
npm run deploy
```
